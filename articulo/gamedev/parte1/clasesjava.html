Clases en Java

Declaracion de clase

Las clases en java constan basicamente de los siguientes elementos:

  • Nombre de la clase
  • Constructor
  • Atributos
  • Metodos

La sintaxis general para definir una clase en Java es la siguiente 

public class Ave {	
			
		//Atributos
		public String color;
		private String sexo;
		private int sexo;

		//Constructor por defecto
		public Ave(){
			color = "negro";
			sexo = "V";
		}

		//Constructor con inicializacion de atributos
		public Ave(String col, String sex){
			color = col;
			sexo = sex;
		}

		//Metodos
		public int getEdad {
			return edad;
		}
		public void setColor(String col) {
			color = col;
		}
		public boolean dormir();
		public int mover();
	}

Declaracion de instancia

Para instanciar un objecto sin atributos iniciales de la clase Ave lo realizamos de la siguiente manera: 

Ave ron = new Ave()

Para instanciar un objecto con atributos utilizando el otro constructor: 

Ave ron = new Ave("blanco","M")
