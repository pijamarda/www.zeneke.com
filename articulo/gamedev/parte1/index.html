Preparacion del entorno en Windows

Descarga

Acceder a la web de Oracle y descargarnos el ultimo JDK disponible

Enlance de descarga

Seleccionar la version JDK para windows, por ejemplo:
"Windows x64 173.08 MB jdk-8u20-windows-x64.exe"

Instalacion

Hacemos doble click sobre el fichero descargado y seguimos las instrucciones

Una vez instalado en windows comprobamos que funciona correctamente.
Lo primero es probar que accedemos al comando javac desde cualquier parte del sistema
Arrancar la consola de Windows Boton windows -> cmd -> Enter y escribir el comando

javac

Deberemos ver los comandos de ayuda para el compilador.
En principio, esto nos indica que esta correctamente instalado

Compilar y ejecutar

Primero vamos a crear nuestro primera programa basico, copiar el codigo indicado a continuacion en un fichero de texto que salvaremos con el nombre HelloWorld.java, es imporante respetar el nombre del fichero y la extension.

public class HelloWorld {
public static void main(String[] args) {
System.out.println("Hello, World");
}
}

Nos situamos en la consola de windows donde esta ubicado el fichero y compilamos con el comando

javac HelloWorld.java

Si ha ido todo bien no nos aparecera ningun mensaje de error, y veremos un fichero generado con nombre HelloWorld.class

Ejecutamos el programa que acabamos de crear. Es imporante no poner ninguna extension detras del nombre

java HelloWorld

Si todo ha ido bien deberemos ver por consola el mensaje "Hello, World"
