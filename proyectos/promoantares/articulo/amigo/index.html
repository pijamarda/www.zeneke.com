Aplicacion de Amigo Invisible

La aplicacion sirve para repartir de manera anónima y por correo electrónico un sorteo de tipo Amigo Invisible

Hace tiempo estuve practicando Python utilizando la antigua API de Google GEARS, y por lo que veo sigue funcionando

Enlace a la aplicacion Amigo Invisible

Como se utiliza:

  1. Seleccionamos el numero de participantes (minimo 3, maximo 12)
  2. Ponemos los nombres de las personas y sus correos
  3. Por ultimo podemos seleccionar un Administrador, este administrador recibira la lista completa. A efectos de comprobar que las asignaciones son correctas o por si alguien ha tenido algun problema al no recibir el correo
